# 实现微软正版登录系统

## 1. 概述

实现微软正版登录系统，允许用户使用微软账号登录Minecraft，获取正版令牌并启动游戏。

## 2. 实现步骤

### 2.1 创建MicrosoftAuthService服务

创建一个新的服务类`MicrosoftAuthService`，用于处理微软登录的所有步骤：

- 设备代码流登录
- Xbox Live身份验证
- XSTS身份验证
- Minecraft访问令牌获取
- 游戏拥有情况检查
- 玩家UUID和用户名获取

### 2.2 修改启动页面

在`启动Page.xaml`中添加一个测试按钮，用于触发微软登录流程：

- 添加"微软正版登录测试"按钮
- 按钮点击事件绑定到新的命令

### 2.3 修改启动ViewModel

在`启动ViewModel.cs`中添加：

- 新的命令`TestMicrosoftAuthCommand`
- 调用`MicrosoftAuthService`的方法
- 处理登录结果

### 2.4 实现登录流程

按照wiki内容实现完整的登录流程：

1. **获取设备代码对**：调用`https://login.microsoftonline.com/consumers/oauth2/v2.0/devicecode`获取设备代码和用户代码
2. **引导用户授权**：显示用户代码和验证URI，引导用户在浏览器中授权
3. **轮询授权状态**：调用`https://login.microsoftonline.com/consumers/oauth2/v2.0/token`轮询授权状态
4. **Xbox Live身份验证**：使用访问令牌获取XBL令牌
5. **XSTS身份验证**：使用XBL令牌获取XSTS令牌
6. **获取Minecraft访问令牌**：使用XSTS令牌获取Minecraft访问令牌
7. **检查游戏拥有情况**：调用`https://api.minecraftservices.com/entitlements/mcstore`检查游戏拥有情况
8. **获取玩家信息**：调用`https://api.minecraftservices.com/minecraft/profile`获取玩家UUID和用户名

### 2.5 集成到启动流程

将微软登录集成到现有的游戏启动流程中：

- 当用户关闭离线模式时，使用微软登录获取的令牌启动游戏
- 保存登录状态，避免重复登录

## 3. 代码修改

### 3.1 创建MicrosoftAuthService.cs

```csharp
// 核心服务，处理微软登录流程
public class MicrosoftAuthService
{
    // 设备代码流登录
    public async Task<DeviceCodeResponse> GetDeviceCodeAsync(string clientId);
    
    // 轮询授权状态
    public async Task<TokenResponse> PollAuthorizationStatusAsync(string clientId, string deviceCode);
    
    // Xbox Live身份验证
    public async Task<XboxLiveAuthResponse> AuthenticateWithXboxLiveAsync(string accessToken);
    
    // XSTS身份验证
    public async Task<XstsAuthResponse> AuthorizeWithXstsAsync(string xblToken);
    
    // 获取Minecraft访问令牌
    public async Task<MinecraftAuthResponse> LoginWithXboxAsync(string uhs, string xstsToken);
    
    // 检查游戏拥有情况
    public async Task<EntitlementsResponse> CheckEntitlementsAsync(string minecraftToken);
    
    // 获取玩家信息
    public async Task<ProfileResponse> GetProfileAsync(string minecraftToken);
}
```

### 3.3 修改启动ViewModel.cs

```csharp
// 添加测试微软登录的命令
[RelayCommand]
private async Task TestMicrosoftAuthAsync()
{
    try
    {
        LaunchStatus = "正在测试微软登录...";
        
        // 调用MicrosoftAuthService进行登录测试
        var result = await _microsoftAuthService.TestMicrosoftLoginAsync();
        
        if (result.Success)
        {
            LaunchStatus = $"登录成功！玩家名: {result.Username}, UUID: {result.Uuid}";
        }
        else
        {
            LaunchStatus = $"登录失败: {result.ErrorMessage}";
        }
    }
    catch (Exception ex)
    {
        LaunchStatus = $"登录异常: {ex.Message}";
    }
}
```

## 4. 注意事项

1. **Azure应用注册**：需要在Azure中注册应用程序，获取client_id
2. **API权限**：确保应用程序具有正确的API权限
3. **异步处理**：所有网络请求必须异步处理，避免阻塞UI
4. **错误处理**：妥善处理各种错误情况，提供友好的错误提示
5. **令牌管理**：安全存储和管理获取的各种令牌
6. **用户体验**：提供清晰的登录流程指引
